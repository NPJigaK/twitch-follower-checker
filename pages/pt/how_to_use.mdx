## Autenticação Twitch

- Na primeira vez que acessar, clique no botão `Authenticate with Twitch` para realizar a autenticação no Twitch.  
- As listas exibidas referem-se ao canal do usuário autenticado.

## Verificando Seguidores
### Lista de Seguidores

Exibe uma lista completa de todos os usuários que seguem atualmente o seu canal.

### Novos Seguidores

Exibe uma lista dos usuários que começaram a seguir o seu canal após a última data de verificação.  
A última data de verificação (data de referência) é exibida como `Last checked:` na tela.

### Lista de Unfollows

Exibe uma lista dos usuários que deixaram de seguir o seu canal desde a última data de verificação.  
A última data de verificação (data de referência) é exibida como `Last checked:` na tela.

## Atualizar Data de Verificação

Clique no botão `CHECK DONE` para atualizar a data de referência `Last checked`.  
**Clique apenas quando quiser atualizar a data de referência.**

## Obter Informações Atualizadas

Clique no botão `REFRESH LISTS` para atualizar cada lista com as informações mais recentes.  
Use esta função durante transmissões ou quando desejar verificar novos seguidores em tempo real.

## Busca de Usuários

Use a barra de busca marcada como `Search...` para filtrar ou pesquisar usuários.  
Você pode pesquisar por datas de quando o usuário seguiu, nome de usuário ou outros detalhes.

## Perguntas Frequentes (FAQ)

### Q. Posso gerenciar múltiplos canais?
A. Apenas um canal por navegador é suportado. Para verificar outros canais, utilize outro navegador.  
Caso haja muita demanda através de [Contribuições](../pt/contribute/), poderá ser considerada a adição dessa funcionalidade.

### Q. O que fazer quando o carregamento estiver lento?
A. A obtenção de seguidores pode levar mais tempo caso você tenha muitos seguidores. Aguarde sem fechar o navegador.
