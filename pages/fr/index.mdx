## Qu'est-ce que Twitch Follower Checker ?

**Twitch Follower Checker** est une application web gratuite qui vous permet de vérifier facilement l'état des abonnés de votre chaîne Twitch.
Identifiez rapidement les nouveaux abonnés ainsi que ceux qui se sont désabonnés.

※ Toutes les opérations et le stockage des données se déroulent intégralement dans votre navigateur. Aucune donnée n'est envoyée vers l'extérieur, assurant ainsi votre confidentialité et sécurité.

Vous pouvez accéder à Twitch Follower Checker depuis <Link class={"\_text-primary-600 \_underline \_decoration-from-font [text-underline-position:from-font]"}href="/" target="\_blank" rel="noopener noreferrer">ici</Link> !

## Principales fonctionnalités
- **Authentification OAuth simplifiée** : Connexion sécurisée via l'authentification officielle de Twitch.
- **Gestion automatique des abonnés** : Visualisez en un clin d'œil les abonnés actuels, les nouveaux abonnés et les désabonnés.
- **Mises à jour en temps réel** : Récupérez instantanément les dernières informations sur vos abonnés.
- **Fonction de recherche** : Recherchez facilement des abonnés par nom d'utilisateur ou ID.
- **Environnement sécurisé** : Toutes les données sont stockées localement dans votre navigateur.
